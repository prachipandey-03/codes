## 🚀 How to Run Java Programs

1. Clone the repository

```bash
git clone https://github.com/prachipandey-03/your-repository-name.git
```

2. Open the desired folder.

3. Compile the Java file:

```bash
javac FileName.java
```

4. Run the program:

```bash
java FileName
```
