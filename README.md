## 🚀 How to Run

1. Clone the repository

```bash
git clone https://github.com/your-username/repository-name.git
```

2. Navigate to the desired topic folder.

3. Compile the Java file:

```bash
javac FileName.java
```

4. Run the program:

```bash
java FileName
```
